什么是less

  • less是一个css预编译器，意思是它可以扩展css语言，添加功能如允许变量（variables）、混合（mixins）、函数（functions）和许多其它的技术，让你的css更具维护性，主题性和扩展性。

  • less官网 英文官网 中文官网

  • less写法

传统css写法

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>less写法</title>
  <style>
    .container>ul{
      list-style: none;
    }
    .container>ul>li>a{
      text-decoration: none;
      color: gray;
    }
  </style>
</head>
<body>
  <div class="container">
    <ul>
      <li><a href="">这是一个超链接</a></li>
      <li><a href="">这是一个超链接</a></li>
      <li><a href="">这是一个超链接</a></li>
      <li><a href="">这是一个超链接</a></li>
      <li><a href="">这是一个超链接</a></li>
    </ul>
  </div>
</body>
</html>

less写法

.container>ul>li>a{
  text-decoration: none;
  color: gray;
  ul{
    list-style: none;
  }
}

  • less注释
  1. //：这类注释不会被编译到css文件中去
  2. /**/：这类注释会被编译到css文件中去
