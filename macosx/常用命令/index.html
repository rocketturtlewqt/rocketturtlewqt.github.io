MacOs X常用终端命令

May 1, 2021
1 min read

目录操作

命令 描述 示例
mkdir 创建目录a mkdir a
rmdir 删除空目录b rmdir b
mvdir 移动目录a到目录b mkdir a b
cd 进入到当前目录下的public子目录(返回上级目录则是cd ..) cd public/
pwd 显示当前目录的绝对路径(/Users/wangqintao/Desktop) pwd
ls 显示当前目录的内容 ls

文件操作

命令 描述 示例
cat 终端下打开当前目录下的demo.txt文件 cat demo.txt
more 分页显示文件内容(按空格键切换到下一页，按ctrl+B返回上一页) more demo.txt
cp 复制文件(复制a.txt，并重新命名为b.txt) cp a.txt b.txt
mv 更改文件名(将a.txt重新命名为b.txt) mv a.txt b.txt
file 显示文件类型(demomv.txt: ASCII text) file demomv.txt
open 使用默认的程序打开文件(open .打开当前目录) open demomv.txt
