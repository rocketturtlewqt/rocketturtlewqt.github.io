js语句与js表达式的区别

  • 一个表达式是一个值，可以放在任何需要这个值的地方

  • 下面这些都是表达式

  1. a
  2. a+b
  3. demo(x)
  4. arr.map()
  5. function test() {}
  • 下面这些都是语句
  1. if语句
  2. for循环
  3. switch case
<!DOCTYPE html>

<head>

  <title>jsx语法规则</title>
</head>

  <div id="container"></div>
  <script type="text/babel">
    const data=['Anguler','React','Vue'];
    //1.创建虚拟dom
    const VDOM=(
      <div class='container'>
        <h2>jsx小练习</h2>

              return <li key={index}>{item}</li>
            })
          }
        </ul>
